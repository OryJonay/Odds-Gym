Installation Guide

Dependencies

Being an Open-AI Gym environment, gym is a major dependency. We also use pandas to make it easier to built the data for the environments. Also, the project support Python version 3.6, 3.7 and 3.8.

Installation

To install, simply run:

pip install git+https://github.com/OryJonay/Odds-Gym.git
